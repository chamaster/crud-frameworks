Another CRUD... And a JavaFX client

This is a simple application that expose CRUD operations in a REST interface which is accessed by a JavaFX application. The client uses JavaFX 2 and FXML. You can download the JNLP file below and execute it locally. The source code of this app is in in my github. Notice that we will also create a few other clients using the Java Technology and also a Rich Javascript clients that makes use of the RESTEasy ajax client.
